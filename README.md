# InterviewPrep

A comprehensive repository for coding interview preparation, containing solutions, notes, and utilities in **Python** and **Java**.

## Structure
- `python/` – All Python programs, organised by topic.
- `java/` – Java programs following standard Maven/Gradle structure.
- `scripts/` – Helper scripts to run programs, tests, etc.
- `docs/` – Documentation and cheat sheets (Git, Jira, frameworks).

## Getting Started
- **Python**: `cd python && python3 -m venv venv && source venv/bin/activate`
- **Java**: Ensure Maven is installed (`brew install maven`), then run `mvn compile` from the `java/` folder.

## Technologies Covered
- Python (FastAPI, Selenium, Pytest)
- Java (Spring Boot, JUnit, Cucumber)
- Git & GitHub
- Jira / Agile methodologies
- BDD (Cucumber)
